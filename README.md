# ONISR – Accidents (Shiny Dashboard)

Tableau de bord interactif développé avec **R Shiny** pour l’analyse des accidents
corporels de la route en France (données ONISR – 2023).

## Fonctionnalités
- Indicateurs clés (accidents, tués, blessés, usagers)
- Analyse temporelle (mois, évolution, prévision)
- Comparaisons démographiques (pyramide des âges)
- Relations exposition / gravité
- Cartographie interactive (Leaflet)
- PCA par département

## Données
Sources : ONISR – fichiers `caract`, `lieux`, `vehicules`, `usagers` (2023).
